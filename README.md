# AAC²E

ASCII Schematics for [AC²E](https://github.com/matthschw/ace) circuits. Created
with [AACircuit](https://github.com/Blokkendoos/AACircuit).

## OP0

<p align="center">Generic Amplifier Symbol</p>

``` VDD
             |         
          |\ |         
          | \|
  INP ----+  + 
          |   \
          |    \
    B ----+ op0 >---- OUT
          |    /
          |   /
  INN ----+  +
          | /|
          |/ |
             |
            VSS 
```
